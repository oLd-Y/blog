Chapter 4. Opening Files

Space q q 关闭整个 nvim 窗口

:LazyExtras 打开 mini.files 插件。mini.files 兼容许多 lazyvim 的文本编辑操作。使用 = 进行修改的保存。

再次找到同一个插件按 x 就可以禁用这个插件。
