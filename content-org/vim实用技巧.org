* 普通模式
| 快捷键             | 作用                                  | 补充                                                                                            |
|-------------------+--------------------------------------+------------------------------------------------------------------------------------------------|
| >G                | 移动当前行到文件末尾的 tab 层级           | G 表示从当前行到行尾                                                                              |
| :'<,'normal .     | 为选中范围执行 . 操作                   | '<,'> 表示选中范围, normal 表示在 normal 模式中执行                                                 |
| ,                 | 跳到上次 f/F 查找到的字符                |                                                                                                |
| 10<C-a> / 10<C-x> | 为当前光标所在或之后的数字加上 10 /减去 10 | 1. <C-a>/<C-x> 本身作用是为数字+1 2. <C-a>/<C-x>可以直接跳转到当前光标后方的数字, 省去了一个查找跳转的步骤 |
* 插入模式 
| 快捷键   | 作用                                      | 补充 |
|---------+------------------------------------------+-----|
| <C-o>   | 进入插入-普通模式                           |     |
| <C-o>zz | 可以快速将当前行定位到屏幕中心而不退出 vim 模式 |     |
| <C-r>0  | 粘贴复制到寄存器 0 中的内容(一般为刚刚复制的内容)  |     |

* 插入-普通模式: 在插入模式下执行一次 normal 模式下的动作就返回插入模式
* 复合命令

|--------+------------|

| S      | ^C         |
| I      | ^i         |
| A      | $a         |
| o      | A          |

* 可重复的操作及如何回退
目的 操作 重复 回退
做出一个修改 {edit} . u
在行内查找下一指定字符 f{char}/t{char} ; ,
在行内查找上一指定字符 F{char}/T{char} ; ,
在文档中查找下一处匹配项 /pattern n N
在文档中查找上一处匹配项 ?pattern n N
执行替换 :s/target/replacement & u
执行一系列修改 qx{changes}q @x u

* 操作符 
命令 用途 补充
c 修改
d 删除
y 复制到寄存器
g~ 反转大小写
gu 转换为小写 可以使用 guu 编辑当前行, 而无需完整的 gugu
gU 转换为大写 同上
> 增加缩进
< 减小缩进
= 自动缩进
! 使用外部程序过滤{motion}所跨越的行
* 插入模式下删除文本
按键操作 用途
<C-h> 删除前一个字符（同退格键）
<C-w> 删除前一个单词
<C-u> 删至行首
