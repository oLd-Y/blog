先在 adobe cloud 中调整软件的默认安装位置，然后下载相应的软件，进行**试用**。

在 github 上下载 AdobeGenp-3.4.2-CGP 类似的软件，运行之后，将 path 设置为设置的安装位置，然后 search。

search 完了之后选择 patch，完成了之后，就可以打开相应的软件正常使用了。
