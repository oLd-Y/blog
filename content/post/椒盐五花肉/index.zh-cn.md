## 食材与调料

### 食材

五花肉、小葱、大蒜、花椒、洋葱

### 调料

盐、小茴香、芝麻

## 步骤

### 备菜

| 火候    | 操作                         |
| ----- | -------------------------- |
| 大火    | 将锅烧热。                      |
| -> 中火 | 下入花椒炒至变色。                  |
| 中火    | 下入白芝麻、小茴香至花椒可以用手搌碎，然后倒出备用。 |
| 中火    | 下入盐，                       |


### 制

| 火候    | 操作      |
| ----- | ------- |
| 大火    | 将锅烧热。   |
| -> 小火 | 放入少许底油。 |
|       |         |
