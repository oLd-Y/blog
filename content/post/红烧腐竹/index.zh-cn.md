## 食材与调料

### 食材

腐竹、木耳、小葱、生姜、大蒜、青尖椒、红尖椒、五花肉、

### 调料

盐、

## 步骤

### 备菜

提前烧一壶开水。

腐竹放入碗中，加入少许盐，提前 2 小时用清水泡发，然后捞出控水备用。

木耳提前 2 个小时泡发。

腐竹切段。

生姜切片。

大蒜切片。

水烧开



### 烧制

| 火候    | 操作      |
| ----- | ------- |
| 大火    | 将锅烧热。   |
| -> 小火 | 放入少许底油。 |
|       |         |
