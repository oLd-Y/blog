## 主料、小料、调料

### 主料

鸡腿、香菇、土豆。

### 小料

青辣椒、红辣椒。

### 调料

冰糖、黄豆酱、生抽、蚝油、花生酱。

## 步骤

### 备菜

鸡腿、香菇洗净切成块，土豆切滚刀块，青红椒切片。



### 制

| 火候    | 操作      |
| ----- | ------- |
| 大火    | 将锅烧热。   |
| -> 小火 | 放入少许底油。 |
|       |         |
