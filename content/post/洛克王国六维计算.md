#洛克王国 

# 一般性的公式

```
生命=(round(0.01*(种族值+0.5*个体值)*(50+2*等级)+等级+10))×(1+性格修正)+成长值

除生命外六维=(round[0.01*(种族值+0.5*个体值)*(50+等级)+10]*(1+性格修正)+成长值
```

# pvp简化公式

成长值:在PVP对战时生命固定为100点，其他固定为50点。
精灵固定等级为60级，星级为5星

```
生命=(round(1.7*(种族值+3*初始个体值)+70))×(1+性格修正)+100

除生命外六维=(round(1.1*(种族值+3*初始个体值)+10))*(1+性格修正)+50
```

生命正向性格：(1.7*(x+30)+70)\*1.2+100
其余性格：(1.1*(x+30)+10)\*1.2+50
