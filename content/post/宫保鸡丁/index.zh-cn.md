## 主料、小料、调料

### 主料

鸡腿。

### 小料

大蒜、生姜、大葱、干辣椒。

### 调料

玉米淀粉、白糖、陈醋、生抽、老抽、盐、白胡椒粉、清水。

## 步骤

### 备菜

鸡腿去骨、用纸吸干水分之后切成丁。

往鸡肉中加入半勺盐、2 勺玉米淀粉、2 勺清水，抓拌至黏稠，然后加入少许食用油抓拌均匀。

大蒜、生姜切片，干辣椒切段，大葱切几段葱白。



### 制

| 火候    | 操作      |
| ----- | ------- |
| 大火    | 将锅烧热。   |
| -> 小火 | 放入少许底油。 |
|       |         |
