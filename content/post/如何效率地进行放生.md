#洛克王国 

# 筛特长

1. 勾选慈悲为怀、爱分享特长的精灵，将其全部保留。
2. 特长勾选同乘，个体资质勾选速度，选出一个速度个体值最高的进行标记。
3. 疾行同步骤2。
4. 灵巧同步骤2。

将特长的精灵筛选完毕之后，就按资质从高到低进行pvp精灵及其种公的筛选。

注：以下放生均以培养pvp完美毕业，即个体全10且性格、个体分布全部正确的精灵为目的。

# 放生所有防御增益性格精灵

在保留了有用特长的精灵之后，我们首先选择防御增益的性格，将其中所有的精灵全部放生。

原因是百分比生命基本上在耐久不会弱于百分比防御，但在在泛用性上远远高于百分比防御，因此可以认为生命增益性格永远优于防御性格。所以，除开一些有防御特化技能的精灵可能使用防御增益性格，其余的精灵可首先将防御性格一键放生。

# 对于了不起的天分

挑选可能的毕业精灵以及其他精灵毕业合适的种公。

由于种公不值得浪费道具修改天分，所以只在“了不起的天分”中挑选种公，其余的天分只进行“捡漏”或者一键放生。

具体做法为：

1. 仅设置“了不起的天分”筛选条件，打开放生模式，一个一个排查精灵是否为合适的种公。
2. 如果无法做为种公，则继续查看正面加成是否能用：
	- 能用，则查看其能否低成本养成。可以则保留，否则将其勾上，等待后续放生。
	- 不能用则直接勾上等待后续放生。
3. 筛查完毕之后将勾上的不需要的精灵一键放生即可。
4. 若在这一步中已找到了毕业精灵，则后续的所有精灵都可一键放生。


# 对于相当好的天分/还不错的天分

由于要想当种公不应使用道具修改天分，而“相当好的天分”和“还不错的天分”必定使用一个金钥匙，所以我们只在这两个天分中进行“捡漏”：
- 在“相当好的天分”中，找性格正确且3个个体值不低于(10,8,9)且至少有两条个体正确的精灵。
- 在“还不错的天分”中，找性格正确且2个个体值不低于(10,9)且两条个体的位置都是正确的精灵。

以下为**快速分辨精灵是否需要**的技巧：
- 性格的减益箭头在“生命、物防、魔防”上的性格都不是好性格，可以直接放生。
- 个体加成与减益箭头重合，则该精灵也属于不合格的精灵。此时再检查正面性格是否合适=>毕业的道具数量是否过高，不过关的也直接勾选等待放生。

具体的做法为：
1. 选择需要的增益性格，一个一个挑选是否有可以培养的精灵，有则将其标记。
2. 取消增益性格的筛选，将未被标记的精灵全都放生。

# 对于一般般的天分

经过前面的筛选之后，很大概率可以得到一只毕业或者接近毕业的精灵，如果没有，则说明捕捉的数量不够，需要继续捕捉，因此“一般般的天分”的精灵可以全部一键放生。
