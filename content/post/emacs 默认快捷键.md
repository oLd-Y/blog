date: 2024-05-08

| 快捷键          | 命令            | 作用             |
| ------------ | ------------- | -------------- |
| C-h m        | describe-mode | 列出当前主模式下的可用快捷键 |
| ESC 后松开      | 无             | 默认等效于 Meta 前缀  |
| C-u args 快捷键 | 无             | 为快捷键增加参数       |
