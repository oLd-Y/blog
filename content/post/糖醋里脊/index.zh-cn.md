## 主料、小料、调料

### 主料

里脊肉、

### 小料

白芝麻、小葱、大蒜。

### 调料

玉米淀粉/土豆淀粉、老抽、

## 步骤

### 备菜

里脊肉切片，打上花刀，然后切成小条备用。



### 制

| 火候    | 操作      |
| ----- | ------- |
| 大火    | 将锅烧热。   |
| -> 小火 | 放入少许底油。 |
|       |         |

## 注意事项

1. 挂糊时如果放鸡蛋，则炸好之后容易回软。
2. 糊不能调得太稀，不然挂不上糊。
