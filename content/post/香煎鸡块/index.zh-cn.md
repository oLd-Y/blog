## 食材与调料

### 食材

鸡翅、小葱、生姜。

### 调料

盐、玉米淀粉、白酒、生抽、老抽、蚝油、白糖。

## 步骤

### 备菜

生姜切厚片。

小葱的葱白和葱绿分开切段。

鸡肉用厨房纸擦干，加一点盐、玉米淀粉抓匀，再加点食用油封一下，腌制10分钟。

### 煎制

| 火候    | 操作                              |
| ----- | ------------------------------- |
| 大火    | 将锅烧热，放入少许底油。                    |
| 大火    | 下入鸡块，每一面煎 2 ~ 3 分钟，煎至表皮微焦后捞出备用。 |
| -> 小火 | 下入葱白、姜煸香，鸡块回锅。                  |
| 小火    | 淋入少许白酒，加入少许生抽、老抽、蚝油、白糖调味。       |
| -> 大火 | 翻炒 2 ~ 3 分钟。                    |
| 大火    | 加入剩下的葱绿，翻匀即可出锅。                 |

## 注意事项

1. 腌制之前必须保证鸡块是干的，否则难以入味。
2. 煎制同样需要尽量控干鸡肉的水分，否则容易使锅温降低，鸡翅难以煎熟。
3. 鸡翅可以换成其它带皮鸡肉，如鸡腿等。
