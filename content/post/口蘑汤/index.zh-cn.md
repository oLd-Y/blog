## 食材与用具

1. 主料：口蘑、鸡蛋。
2. 小料：枸杞、小葱/香菜。
3. 调料：盐。
4. 其它：清水。
5. 用具：铁锅、剪刀。

## 步骤

首先是备菜环节。

口蘑把柄去掉。

接着我们开始制作。

| 火候    | 操作      |
| ----- | ------- |
| 大火    | 将锅烧热。   |
| -> 小火 | 放入少许底油。 |
|       |         |
