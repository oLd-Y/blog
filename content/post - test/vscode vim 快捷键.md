# vim-surround

| 快捷键   | 作用                     |
| ----- | ---------------------- |
| S"    | 为选中的内容增加双引号            |
| ysiw[ | 选中单词并增加中括号(左括号会额外增加空格) |
| cs"'  | 将当前所在最近的双引号改为单引号       |
| ds"   | 删除选中内容的双引号             |
